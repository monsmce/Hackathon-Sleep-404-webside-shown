# **ViaRooms**
Multi-level system combining React, C# and .NET WebApi to identify and display concurrent occupancy states.

_extra_: a simple [React mapping tool](https://github.com/MrPAkaCannyBoiz/Hackathon-Sleep-404-team/tree/main/src/VIARooms.Mapper) to facilitate registration for visual representations of the rooms, names and locations on a map.
